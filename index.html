Introduction

Welcome to my CS 185 PR1!

My name is Anthony Hall, and I am a fourth-year CS major. Ever since taking computer graphics (CS 180) here last year, I have found a bit of a passion for writing procedural shaders on Shadertoy.

What is a shader? A shader can be thought of as a program that several pixels individually run in parallel given some inputs (e.g. position) to generate an output color. So everything you find on this site is exactly that - images generated by code (and sometimes external assets such as images).

If any of that sounds exciting, be sure to navigate to the other pages to find a shader gallery, useful videos, and links to my own projects!
